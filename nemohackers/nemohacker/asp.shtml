Well first of all, ASP stands for Active Server Pages. ASP is a server side technology, this means it works on any web browser, because all the work is done at the web server end. Essentially ASP pages are just normal HTML with scripts embedded in them. You can write your scripts in VBScript, Javascript or any language which is ActiveScript compatible.
