Linux

Linux is the operating system of the future. When first making the choice of running linuxmost people forget that it is free. Unlike it's predisor UNIX, which was far too expensive for theaverage users, Linux is a free Operating System distruted on the Open Source Project. There are may different Distrobutions of Linux of which each differ from on one level or another.
Linux was created by a man called Linus Torvald, a programmer from Finland, who in 1991 completed thefirst release of Linux. Since then there has been a huge community of Linux Gurus, lovers, and newbies.This section aims to help you get into this OS revolution.
